# DropShare — Private Local File Sharing Web Server

DropShare is a modern local file sharing web server designed for fast, secure, and cross-platform file transfer between devices on the same WiFi network. It allows users to share photos, videos, documents, ZIP files, APKs, PDFs, and other file types directly between Android phones, iPhones, and Windows laptops through a clean browser-based interface.

The system works as a private local web server running on a personal computer using Node.js and Express.js. Devices connected to the same network can access the server through a browser without requiring cloud storage, external hosting, or account creation.

The interface uses a professional dark theme with a black background, glass-style UI elements, smooth animations, responsive layouts, and real-time transfer progress. The design focuses on a modern expert-level appearance rather than a colorful or childish style.

Main Features:

* Cross-platform support (Android, iPhone, Windows)
* Share all file types
* Drag and drop upload system
* Real-time transfer progress
* QR code connection support
* Responsive mobile-friendly design
* Local network transfer for high speed
* Private and secure LAN-based access
* No account or login required
* Browser-based access with no installation on phones

Technologies Used:

Frontend:

* React
* Next.js
* Tailwind CSS
* Framer Motion

Backend:

* Node.js
* Express.js
* Socket.IO
* Multer

How to run:
* Make sure it runs on port number 3000 on your IP Address
* You can also just click the exe file if running windows os, it will manage everything automatically and show web link 

Purpose:

The project is designed as a personal private file transfer system similar to AirDrop or SHAREit but focused on local web-based sharing with a professional modern UI and full control over the server.
